## 第 1 章 前置要求

### 1.1 系统要求

**操作系统：**

- Windows 10/11 (推荐)
- macOS 10.15+
- Linux Ubuntu 18.04+

**硬件要求：**

- CPU: 4 核以上
- 内存：8GB 以上（推荐 16GB）
- 硬盘：至少 20GB 可用空间

### 1.2 软件要求

**必须安装：**

1. **Node.js** (>= 18.x)

   ```bash
   # Windows/macOS
   下载安装包：https://nodejs.org/

   # 验证安装
   node --version
   npm --version
   ```

2. **Python** (>= 3.9)

   ```bash
   # Windows
   下载安装包：https://www.python.org/downloads/
   安装时勾选"Add Python to PATH"

   # macOS (使用 Homebrew)
   brew install python@3.9

   # 验证安装
   python --version
   pip --version
   ```

3. **Git**

   ```bash
   # Windows
   下载安装包：https://git-scm.com/download/win

   # macOS
   brew install git

   # 验证安装
   git --version
   ```

4. **VSCode** (推荐编辑器)

   ```
   下载地址：https://code.visualstudio.com/

   推荐插件:
   - Vue Language Features (Volar)
   - Python
   - Prettier
   - ESLint
   ```

## 第 3 章 后端环境搭建

### 3.1 创建虚拟环境

```bash
# 进入后端目录
cd PMP_Service

# 创建虚拟环境（Windows）
python -m venv venv

# 激活虚拟环境（Windows PowerShell）
.\venv\Scripts\Activate.ps1

# 激活虚拟环境（macOS/Linux）
source venv/bin/activate
```

### 3.2 安装 Python 依赖

```bash
# 升级 pip
python -m pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

**requirements.txt 内容：**

```txt
fastapi==0.104.1
uvicorn[standard]==0.24.0
sqlalchemy==2.0.23
python-dotenv==1.0.0
pydantic==2.5.0
aiosqlite==0.19.0
langchain==0.1.0
langchain-community==0.0.10
python-jose[cryptography]==3.3.0
passlib[bcrypt]==1.7.4
python-multipart==0.0.6
```

### 3.3 配置环境变量

```bash
# 复制环境变量示例文件
cp .env.example .env

# 编辑.env 文件
# Windows: 用记事本打开
# macOS/Linux: vim .env
```

**.env 文件内容：**

```ini
# ==================== 数据库配置 ====================
DATABASE_TYPE=sqlite
SQLITE_DB_PATH=database.db

# ==================== JWT 配置 ====================
JWT_SECRET_KEY=your-secret-key-here-change-in-production
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440

# ==================== CORS 配置 ====================
FRONTEND_URL=http://localhost:5173

# ==================== AI 配置 ====================
DEFAULT_AI_MODEL=tongyi

# AI Token 配置（后续在页面配置）
# DASHSCOPE_API_KEY=your-dashscope-key
# WENXIN_API_KEY=your-wenxin-key
# KIMI_API_KEY=your-kimi-key
```

### 3.4 初始化数据库

```bash
# 运行数据库初始化脚本
cd ..
python scripts/init_db.py

# 成功后会看到:
# ✓ 数据库初始化完成
# ✓ 表已创建
# ✓ 默认管理员账号已创建
```

### 3.5 启动后端服务

```bash
# 回到后端目录
cd PMP_Service

# 启动开发服务器
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 成功启动后看到:
# INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
# INFO:     Application startup complete.
```

**访问 API 文档：**

```
浏览器打开：http://localhost:8000/docs
```

## 第 7 章 常见问题

### 7.1 依赖安装失败

**问题：** pip install 报错

**解决方案：**

```bash
# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或手动安装每个依赖
pip install fastapi uvicorn sqlalchemy ...
```

### 7.2 端口被占用

**问题：** 端口 8000 或 5173 已被占用

**解决方案：**

```bash
# Windows 查看占用端口的进程
netstat -ano | findstr :8000
taskkill /F /PID <PID>

# 修改配置文件
# 后端：uvicorn app.main:app --port 8001
# 前端：vite.config.js 中修改 port
```

### 7.3 数据库连接失败

**问题：** SQLAlchemy 无法连接数据库

**解决方案：**

```bash
# 检查.env 配置
cat .env

# 确保 DATABASE_TYPE=sqlite
# SQLite 路径正确

# 重新初始化数据库
python scripts/init_db.py
```

### 7.4 AI 连接失败

**问题：** AI 模型连接超时

**解决方案：**

```
1. 检查 Token 是否正确
2. 检查网络连接
3. 更换其他 AI 模型测试
4. 查看后端日志
```

### 7.5 前端页面空白

**问题：** 打开页面白屏

**解决方案：**

```
1. 打开浏览器开发者工具
2. 查看 Console 错误信息
3. 检查 Network 请求是否成功
4. 确认后端 API 正常运行
```
