### 1.2 注册与登录

#### 1.2.1 账号注册

**步骤 1：访问注册页面**

```
1. 打开浏览器，访问系统地址（如：http://localhost:3000）
2. 点击登录页面的"注册账号"按钮
```

**步骤 2：填写注册信息**

```
需要填写的信息：
□ 用户名：3-20 个字符，字母开头
□ 邮箱：有效的邮箱地址
□ 密码：8-50 位，包含大小写字母、数字、特殊字符
□ 确认密码：与上面输入的密码一致
□ 验证码：点击获取短信验证码

示例：
用户名：zhangsan
邮箱：zhangsan@example.com
密码：Secure@Pass123
```

**步骤 3：完成注册**

```
1. 阅读并同意用户协议
2. 点击"立即注册"按钮
3. 系统自动登录并跳转到首页
```

#### 1.2.2 登录系统

**账号密码登录：**

```
1. 打开登录页面
2. 输入用户名和密码
3. （可选）勾选"记住我"（7 天内免登录）
4. 点击"登录"按钮
5. 成功登录后跳转到 Dashboard
```

**忘记密码：**

```
1. 点击登录页面的"忘记密码"链接
2. 输入注册时的邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接
5. 设置新密码
6. 使用新密码登录
```

### 1.3 创建第一个项目

#### 1.3.1 手动创建项目

**详细步骤：**

**步骤 1：进入新建页面**

```
1. 登录后进入系统首页
2. 点击顶部导航栏的"项目"菜单
3. 点击右上角的"+ 新建项目"按钮
```

**步骤 2：填写基本信息**

```
必填字段：
• 项目名称 *：例如"电商平台重构项目"
• 项目描述 *：简要介绍项目（10-2000 字）
• 项目类型 *：从下拉菜单选择
  - 新产品开发
  - 功能迭代
  - 技术重构
  - Bug 修复专项
  - 系统优化
• 优先级 *：选择优先级
  - P0（紧急且重要）
  - P1（重要不紧急）
  - P2（紧急不重要）
  - P3（常规任务）
• 开始日期 *：项目计划开始时间
• 结束日期 *：项目计划结束时间
```

**步骤 3：填写详细信息（可选）**

```
可以填写以下信息让项目更完整：
• 项目背景：为什么要做这个项目
• 目标用户：谁会使用这个产品
• 业务价值：项目带来的价值
• 预期成果：可衡量的指标
• 预算：项目预算金额
```

**步骤 4：保存项目**

```
1. 检查填写的信息是否正确
2. 点击"保存并创建"按钮
3. 系统提示"创建成功"
4. 自动跳转到项目详情页
```

#### 1.3.2 使用 AI 智能补全

**什么是 AI 智能补全？**

```
AI 智能补全功能可以根据你提供的简短描述，自动生成完整的项目信息，包括：
✓ 项目背景（500 字左右）
✓ 目标用户（详细描述）
✓ 业务价值（300 字左右）
✓ 预期成果（可衡量指标）

可以节省你 70% 的文档编写时间！
```

**使用步骤：**

**步骤 1：切换到 AI 补全模式**

```
1. 在创建项目页面
2. 找到"创建方式"选项
3. 选择"AI 智能补全"（默认是"手动填写"）
```

**步骤 2：输入简短描述**

```
在输入框中输入一句话描述，例如：

"为中小企业打造的一站式营销管理平台，整合主流营销渠道，提供智能化营销解决方案。"

提示：描述越具体，AI 生成的内容越准确。
```

**步骤 3：等待 AI 生成**

```
1. 点击"开始 AI 补全"按钮
2. 系统显示 AI 思考过程（约 5-10 秒）
3. AI 自动生成完整的项目信息
```

**步骤 4：审查和修改**

```
AI 生成的内容会填充到各个字段，你可以：
✓ 直接接受：如果满意 AI 生成的内容
✓ 手动修改：点击任意字段进行修改
✓ 重新生成：如果不满意，可以点击"重新生成"
✓ 继续对话：通过对话让 AI 进一步完善
```

**步骤 5：保存项目**

```
1. 确认所有信息正确
2. 点击"保存并创建"
3. 完成项目创建
```

### 1.4 邀请团队成员

#### 1.4.1 添加项目成员

**步骤 1：进入成员管理**

```
1. 打开项目详情页
2. 点击左侧菜单的"团队成员"
3. 点击右上角的"+ 添加成员"按钮
```

**步骤 2：搜索并选择成员**

```
可以通过以下方式查找成员：
• 搜索用户名
• 搜索邮箱
• 从组织架构中选择
• 输入对方邮箱邀请新人
```

**步骤 3：设置角色**

```
为每个成员分配合适的角色：
□ 项目负责人：拥有项目的全部权限
□ 技术负责人：负责技术决策和 Code Review
□ 产品经理：负责需求管理和验收
□ 开发工程师：负责任务开发
□ 测试工程师：负责测试和质量
□ 观察者：只能查看，不能编辑

提示：一个用户可以有多个角色
```

**步骤 4：发送邀请**

```
1. 点击"发送邀请"按钮
2. 被邀请人会收到邮件通知
3. 接受邀请后成为项目成员
```

#### 1.4.2 管理成员权限

**查看成员列表：**

```
在项目详情页 -> 团队成员，可以看到：
• 所有成员的头像、姓名、角色
• 加入项目的时间
• 当前负荷情况
• 负责的任务数量
```

**修改成员角色：**

```
1. 找到要修改的成员
2. 点击右侧的"编辑"按钮
3. 修改角色
4. 点击"保存"
```

**移除成员：**

```
1. 找到要移除的成员
2. 点击右侧的"移除"按钮
3. 确认移除操作

注意：
• 不能移除项目负责人
• 移除后该成员将失去项目访问权限
• 已分配的任务会自动释放
```

### 2.1 项目列表管理

#### 2.1.1 查看项目列表

**访问项目列表：**

```
1. 点击顶部导航栏的"项目"菜单
2. 进入项目列表页面
```

**列表视图：**

```
项目列表以卡片形式展示，每个卡片显示：
• 项目名称
• 项目描述（前 50 字）
• 优先级标签（P0/P1/P2/P3）
• 进度条（百分比）
• 健康度评分（星级）
• 团队成员头像
• 最后更新时间
```

#### 2.1.2 筛选和搜索

**按条件筛选：**

```
可以使用以下筛选条件：
□ 项目状态：全部/草稿/已立项/进行中/已完成/已归档
□ 优先级：全部/P0/P1/P2/P3
□ 项目类型：全部/新产品开发/功能迭代/技术重构...
□ 我的项目：只看我负责的项目
□ 我参与的项目：看我参与的所有项目
```

**搜索项目：**

```
在搜索框中输入关键词，可以搜索：
• 项目名称
• 项目描述
• 负责人姓名

支持模糊搜索，例如输入"电商"可以找到所有与电商相关的项目。
```

**组合筛选示例：**

```
场景：查找我负责的进行中的 P0 优先级项目

操作步骤：
1. 勾选"我的项目"
2. 状态选择"进行中"
3. 优先级选择"P0"
4. 系统自动显示符合条件的结果

结果：实时筛选，无需刷新页面
```
