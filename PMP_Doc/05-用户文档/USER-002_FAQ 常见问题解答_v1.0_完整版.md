## 第 1 章 账号与登录

### 1.1 注册相关问题

#### Q1.1: 注册时收不到验证码怎么办？

**问题描述：**
在注册页面输入手机号后，点击"获取验证码"按钮，但是没有收到短信验证码。

**可能原因：**

1. 手机号输入错误
2. 手机信号不好或欠费
3. 短信服务商延迟
4. 验证码发送过于频繁

**解决方案：**

✅ **检查手机号**

```
1. 确认手机号格式正确（11 位数字）
2. 确认国家/地区代码选择正确
3. 重新输入手机号后再次尝试
```

✅ **等待并重试**

```
1. 等待 60 秒后重新获取验证码
2. 如果连续发送 3 次，需要等待 24 小时
3. 建议先检查手机是否有信号
```

✅ **检查手机设置**

```
1. 查看是否开启了短信拦截功能
2. 检查垃圾短信箱
3. 确认手机没有欠费停机
```

✅ **联系客服**

```
如果以上方法都无效，请联系客服：
• 在线客服：点击右下角"在线咨询"
• 邮箱支持：support@ai-agent-pm.com
• 工作电话：400-XXX-XXXX（工作日 9:00-18:00）
```

**预防措施：**

- 使用常用手机号注册
- 确保手机信号良好
- 不要频繁发送验证码

---

#### Q1.2: 用户名已存在怎么办？

**问题描述：**
注册时提示"用户名已被注册"，无法继续。

**解决方案：**

✅ **更换用户名**

```
建议使用以下组合：
• 姓名 + 数字：zhangsan2024、wangwu888
• 姓名 + 字母：zhangsanzs、lisi ls
• 创意用户名：developer_zhang、product_li
```

✅ **使用邮箱登录**

```
如果喜欢的用户名被占用，可以：
1. 使用其他用户名注册
2. 登录后绑定邮箱
3. 以后使用邮箱登录
```

✅ **找回旧账号**

```
如果是你以前注册的账号：
1. 点击"忘记密码"
2. 通过邮箱重置密码
3. 重新登录即可
```

**用户名命名规则：**

```
✓ 允许：
• 3-20 个字符
• 字母开头
• 可包含字母、数字、下划线
• 区分大小写

✗ 不允许：
• 纯数字
• 特殊字符（!@#$%^&*等）
• 敏感词汇
• 已存在的用户名
```

### 1.2 登录相关问题

#### Q1.3: 忘记密码如何找回？

**问题描述：**
忘记登录密码，无法访问系统。

**找回密码步骤：**

**方法一：邮箱找回（推荐）**

```
步骤 1：进入找回密码页面
  • 打开登录页面
  • 点击"忘记密码"链接

步骤 2：输入注册邮箱
  • 输入你注册时使用的邮箱地址
  • 点击"发送重置邮件"

步骤 3：查收邮件
  • 打开邮箱（检查垃圾邮件箱）
  • 找到标题为"重置密码"的邮件
  • 邮件有效期为 30 分钟

步骤 4：重置密码
  • 点击邮件中的重置链接
  • 输入新密码（8-50 位）
  • 确认新密码
  • 点击"保存"

步骤 5：重新登录
  • 返回登录页面
  • 使用新密码登录
```

**方法二：手机短信找回**

```
步骤 1：选择短信验证
  • 在找回密码页面
  • 切换到"手机验证"选项卡

步骤 2：输入手机号
  • 输入注册时的手机号
  • 点击"获取验证码"

步骤 3：验证身份
  • 输入收到的短信验证码
  • 点击"下一步"

步骤 4：设置新密码
  • 输入新密码
  • 确认新密码
  • 完成重置
```

**方法三：联系管理员**

```
如果以上方法都不可用：
1. 联系你们公司的系统管理员
2. 提供你的身份证明
3. 管理员可以重置你的密码
4. 首次登录后立即修改密码
```

**安全提示：**

```
⚠️ 注意事项：
• 重置链接 30 分钟后失效
• 不要告诉任何人你的密码
• 不要在公共电脑上记住密码
• 定期更换密码（建议 90 天）
• 启用双重认证更安全
```

#### Q1.4: 登录后提示"Token 已过期"怎么办？

**问题描述：**
使用过程中突然弹出提示"登录凭证已过期，请重新登录"。

**原因说明：**

```
为了安全考虑，系统设置了登录有效期：
• 普通登录：2 小时有效
• 记住我登录：7 天有效
• 超过有效期需要重新登录
```

**解决方案：**

✅ **自动跳转登录**

```
系统会自动：
1. 保存你当前的操作
2. 跳转到登录页面
3. 登录后自动返回原页面
4. 继续未完成的操作
```

✅ **手动刷新**

```
如果没有自动跳转：
1. 点击提示框的"确定"按钮
2. 手动刷新页面（F5）
3. 重新登录
```

✅ **防止过期的方法**

```
• 勾选"记住我"（7 天内免登录）
• 每 1 小时有一次页面操作
• 保持浏览器开启状态
• 不要清除浏览器缓存
```

**长时间不操作的情况：**

```
场景：填写表单时间过长导致过期

解决：
1. 系统会提示"会话已过期"
2. 点击"重新登录"
3. 登录后表单内容还在
4. 可以继续填写并提交
```

### 1.3 密码管理

#### Q1.5: 如何修改登录密码？

**修改密码步骤：**

**步骤 1：进入设置页面**

```
1. 登录系统
2. 点击右上角头像
3. 选择"个人设置"
4. 点击"账户安全"标签
```

**步骤 2：验证身份**

```
需要验证当前密码：
• 输入当前登录密码
• 点击"验证"
• 验证通过后才能修改
```

**步骤 3：设置新密码**

```
输入新密码要求：
✓ 长度：8-50 位
✓ 包含：大写字母 + 小写字母 + 数字 + 特殊字符
✓ 不能：与前 5 次密码相同
✓ 建议：使用密码管理器生成强密码

示例密码：
• Secure@Pass2024 ✓
• MyP@ssw0rd#123 ✓
• Simple123 ✗（太简单）
```

**步骤 4：确认修改**

```
1. 再次输入新密码（确认）
2. 点击"保存修改"
3. 系统提示"修改成功"
4. 需要使用新密码重新登录
```

**修改成功后：**

```
• 所有设备会被强制登出
• 需要使用新密码重新登录
• 修改记录会保存在安全日志中
• 如有异常会收到邮件通知
```

**密码修改失败的原因：**

```
❌ 当前密码输入错误
  → 请仔细检查后重试

❌ 新密码不符合要求
  → 按照密码规则重新设置

❌ 新密码与旧密码相同
  → 必须使用不同的密码

❌ 两次输入的新密码不一致
  → 请仔细核对后重试
```

#### Q2.2: 项目名称重复了怎么办？

**问题描述：**
创建项目时提示"项目名称已存在"。

**原因：**

```
同一组织内，项目名称必须唯一。
例如：已经有一个"电商平台"项目，
就不能再创建同名的项目。
```

**解决方案：**

✅ **方案一：修改项目名称**

```
在原名基础上添加修饰词：
• 电商平台（2024 版）
• 电商平台 - 移动端
• 电商平台重构项目
• 电商平台二期
```

✅ **方案二：使用前缀/后缀**

```
• 【新项目】电商平台
• 电商平台_v2.0
• 电商平台（升级款）
```

✅ **方案三：使用项目编码**

```
如果有项目编号习惯：
• PM-2024-001 电商平台
• PM-2024-002 电商平台升级
```

**查找重复项目：**

```
1. 在项目列表搜索同名项目
2. 查看项目详情和状态
3. 如果是已完成/已归档项目
   可以考虑：
   • 复用旧项目（复制功能）
   • 在旧项目上继续
   • 使用不同的名称
```

**预防重名的小技巧：**

```
• 建立项目命名规范
  例如：部门 - 类型 - 名称 - 版本

• 使用日期区分
  例如：电商平台_2024Q1

• 加入团队标识
  例如：A 组 - 电商平台
```

```

2. 点击项目名称或"查看详情"按钮

```
可以修改的内容：
✓ 基本信息：名称、描述、类型、优先级

✓ 详细信息：背景、目标、价值等

不能修改的内容：

```
1. 点击"保存"按钮

```

## 第 3 章 文档管理

### 3.1 AI 对话使用

#### Q3.1: AI 对话如何使用？

**什么是 AI 对话？**

```
AI 对话功能让你通过与 AI 助手聊天来生成文档。
就像有一个专业的文档编写助手在帮助你。

适用场景：
• 编写需求文档（PRD）
• 编写设计文档
• 编写技术方案
• 编写会议纪要
• 任何文字工作
```

**使用步骤：**

**步骤 1：打开 AI 对话**

```
两种打开方式：
方式一：一键生成
  1. 进入文档管理页面
  2. 点击"+ 新建文档"
  3. 选择"AI 辅助生成"
  4. AI 对话抽屉自动打开

方式二：主动对话
  1. 点击页面右下角的"AI 助手"悬浮按钮
  2. AI 对话抽屉从右侧滑出
```

**步骤 2：描述你的需求**

```
在对话框中输入你的要求，例如：

"帮我写一份电商平台的需求文档，包括：
- 用户端：商品浏览、购物车、下单支付
- 商家端：商品管理、订单管理、数据统计
- 管理后台：用户管理、内容审核、系统配置"

提示：描述越详细，AI 生成的内容越准确。
```

**步骤 3：与 AI 多轮对话**

```
AI 会生成初稿，你可以：

继续完善：
"请细化支付流程的业务规则"

修改调整：
"把用户注册流程改得更简单一些"

补充内容：
"增加安全性方面的考虑"

AI 会记住上下文，理解你的意图。
```

**步骤 4：保存文档**

```
对生成的内容满意后：
1. 点击"保存为新文档"按钮
2. 填写文档标题和版本号
3. 选择文档类型
4. 点击"保存"
5. 文档保存到系统中
```

**使用技巧：**

✅ **提供足够的上下文**

```
好的示例：
"为中小企业 CRM 系统写一份用户注册登录模块的
需求文档，需要支持手机号注册、微信登录、以
及企业微信第三方登录。"

差的示例：
"写一个登录功能"
→ 太简单，AI 无法理解具体需求
```

✅ **分步骤引导 AI**

```
第一轮："生成文档大纲"
第二轮："展开第一章内容"
第三轮："补充流程图说明"
第四轮："添加验收标准"

逐步完善，比一次性要求更好。
```

✅ **使用预设模板**

```
系统提供了常用模板：
• 需求文档模板
• 设计文档模板
• 技术方案模板
• 会议纪要模板

点击"模板"按钮，选择合适的模板，
AI 会按照模板结构生成内容。
```

#### Q3.2: AI 生成的内容不满意怎么办？

**常见原因：**

❌ **原因 1：描述不够具体**

```
问题："帮我写个文档"
解决：改为"帮我写一份电商平台的用户管理模块需求文
档，包括注册、登录、个人信息管理三个功能"
```

❌ **原因 2：缺少业务背景**

```
问题：直接让 AI 生成
解决：先介绍业务背景："我们是一个 B2B 电商平台，主
要服务中小批发商..."
```

❌ **原因 3：没有多轮对话**

```
问题：期望一次就完美
解决：通过多轮对话逐步完善："第一版不错，但需要增
加..."
```

**改进方法：**

✅ **方法一：提供更多细节**

```
添加以下信息：
• 目标用户群体
• 核心业务流程
• 特殊要求
• 参考案例

示例：
"这是一个面向年轻人的社交 APP，主要功能包括...
参考微信和 QQ 的部分功能，但要更简洁..."
```

✅ **方法二：使用示例引导**

```
给 AI 提供参考：
"类似淘宝的商品详情页，包括：
- 商品图片和视频
- 价格和库存显示
- 用户评价
- 店铺信息
- 客服入口"
```

✅ **方法三：分段生成**

```
不要一次性生成整篇文档，而是：
1. 先生成大纲
2. 逐章生成内容
3. 每章都可以修改调整
4. 最后整合
```

✅ **方法四：人工修订**

```
AI 生成后：
1. 通读全文，标记需要修改的地方
2. 直接在编辑器中修改
3. 或者通过对话让 AI 修改：
   "第二段需要更详细"
   "这个流程图不对，应该是..."
```
