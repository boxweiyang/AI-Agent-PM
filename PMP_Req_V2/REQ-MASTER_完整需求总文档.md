## 2. 建议全流程顺序（产品视角）

```
登录后 **应用壳与首页(M09)** → 进入项目 → 业务协作 **M01～M08**；**通用规范与验收** 见 **M12**。

立项(M01) → 需求与模块文档(M02) → 技术与接口/库设计(M02B/M02C/M02D)
         → 迭代与 Story(M03) → **人力池与预订(M05)** → **Task 分配与执行(M04)**
         → **提测/缺陷/联调(M07)** → **Dashboard/管理分析(M08)** → **收尾/复盘/知识(M10)** → …

并行（全程）：**变更请求 CR(M06)**；**全站 AI 交互(M11)** 贯穿各模块。
```
